# @dbland/ui Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-15

### Added

- Base UI components (24 components)
  - Alert, AlertDialog, Avatar, Badge, Button, Card
  - Checkbox, Collapsible, ContextMenu, Dialog, DropdownMenu
  - Input, Label, Popover, ScrollArea, Select, Separator
  - Sheet, Skeleton, Switch, Table, Tabs, Textarea, Tooltip
- Radix UI primitives integration
- Tailwind CSS styling
- `cn()` utility for class merging
